JavaScript随机数

  • Javacript
  • 前端相关

随机数在很多场景下都会用到，比如说抽奖。对于前端来说，随机数也是常用的一种手段，比如说验证码生成等。如何生成有效、可控范围内的随机数呢？本文将介绍一下Javascript随机数生成。

Javascript random()方法

该语法返回的是0.0~1.0之间的一个伪随机数，但不包括0和1两个数。那么如何生成指定范围内的随机数呢？在解决这个问题之前，我们先需要介绍一下Javascript的取整及舍入的方法。

Javascript 取整、舍入

对x进行下舍入，即小于等于x的最大整数。x可以为任意的数值或表达式。比如x=1.6，则该表达式结果为1。

对x进行上舍入，即大于等于x的最大整数。x可以为任意数值或表达式。

对x进行四舍五入，并取其最接近的整数。x可以为任意数值或表达式。比如x=4.4，则该表达式结果为4。

以上三个表达式是在随机数中经常会用到的三种舍入方法，我们可以通过这三种方法与random方法的配合生成指定范围内的随机数。
