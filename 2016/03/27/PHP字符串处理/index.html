  • PHP
  • 字符处理
  • 教程

有人说编程的早期，甚至整个编程生涯，都是在处理字符串。合格的菜鸟要能合格的处理字符串。我是菜鸟，我要处理字符串。

移除指定字符

  • string trim(string str, string desStr)：移除字符串str两侧的desStr；
  • string ltrim(string str, string desStr)：移除字符串str左侧的desStr；
  • string rtrim(string str, string desStr)：移除字符串str右侧的desStr；

  1. ‘\0’：NULL；
  2. ‘\t’：制表符；
  3. ‘\n’：换行；
  4. ‘\x0B’：垂直制表符；
  5. ‘ ‘：空格；
  6. ‘\r’：回车；

格式化输出

  • string sprintf(string format[,mixed args..])：格式化输出，并将结果返回；
  • void printf(string format[,mixed args…])：格式化输出，直接将结果打印（与echo一致）；

注：format中的类型可以有以下几种：

  1. b：二进制整数；
  2. c：字符整数；
  3. d：单精度浮点数；
  4. f：双精度浮点数；
  5. o：八进制整数；
  6. s：字符串；
  7. u：非指定小数；
  8. x：小写字符a-f的十六进制整数；
  9. X：大写字符A-F的十六进制整数；

这种情况下如果希望打印’%’符号，则需要使用两个’%%’表示。

  • string ucfirst(string str)：如果str第一个字符是字母，则将其转换成大写；
  • string ucwords(string str)：将str中每个单词第一个字母转换成大写；

字符串排序

  • int strcmp(string str1, string str2)：区分大小写方式比较str1与str2；相等返回0，str1在str2后面则返回正数；
  • int strcasecmp(string str1, string str2)：不区分大小写的方式比较str1与str2；相等返回0，str1在str2后面则返回正数；
  • int strnatcmp(string str1, string str2)：按照习惯将str1与str2排序，不区分大小写；

字符串查找

  • int stripos(string str, string substr)：返回substr在str中第一次出现的位置，大小写不敏感；
  • int strpos(string str, string substr)：返回substr在str中第一次出现的位置，大小写敏感；
  • int strripos(string str, string substr)：返回substr在str中最后出现的位置，大小写不敏感；
  • int strrpos(string str, string substr)：返回substr在str中最后出现的位置，大小写敏感；
  • string stristr(string str, string curstr)：返回curstr在str中第一次出现位置之后的部分，大小写不敏感；
  • string strstr(string str, string curstr)：返回curstr在str中第一次出现位置之后的部分，大小写敏感；
